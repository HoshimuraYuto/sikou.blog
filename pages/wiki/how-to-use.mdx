# 使い方

このサイトの使い方。

## コンポーネント

当サイトでは、サイトコンテンツを見やすくするために以下のコンポーネントを作成しています。

### プレビューリンク

一部のリンクは`meta`キーを押したままリンクにカーソルを合わせると、リンク先のコンテンツを閲覧できます。

Windows の場合は`⊞`、Mac の場合は`⌘`を押したまま、リンクにカーソルを合わせます。

import { PreviewLink, Checkbox } from "@/components";

例: <PreviewLink href="/about">About ページ</PreviewLink>をプレビュー

### チェックボックス

変更できるチェックボックスを作成できます。

<br />

例:

<br />

<Checkbox checked={true}>最初からチェックされている</Checkbox>

<Checkbox>チェックがない</Checkbox>
